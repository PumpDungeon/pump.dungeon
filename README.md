# Pump Dungeon

## Usage

```bash
git clone https://github.com/PumpDungeon/pump.dungeon
cd pump.dungeon
git submodule update --init --recursive --remote
docker compose up -d
```

You can now access the website at [http://localhost](http://localhost)
